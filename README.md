# Globe Cads

### Demo

https://globe-cards-app.onrender.com

## Tech Stack

**Client:** React, Typescript, GraphQL & RESTful

**Server:** Node, Express, MongoDB

## Installation

Install the project with npm

```bash
./client
  npm install 
  npm start
```
```bash
./additional-service
  npm install 
  npm start
```

### My API - Get all country data
```http
  GET - https://globe-cards-api.onrender.com
```
